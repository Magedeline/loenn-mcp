loenn-mcp v3.0.0
================

Closes the styleground / trigger gap reported by users — previously you could
add entities and rooms via MCP, but stylegrounds and triggers required opening
Lönn manually. Now everything is reachable from the agent.


WHAT'S NEW IN v3.0
------------------

6 new MCP tools (28 tools total):

  Triggers (room-level, mirror add_entity / remove_entity):
    add_trigger             — place a trigger in a room with width/height
                              and optional path nodes; auto-assigns a unique
                              ID across both entities and triggers.
    remove_trigger          — delete a trigger by ID.

  Stylegrounds (map-level, under Style/Foregrounds and Style/Backgrounds):
    list_stylegrounds       — numbered listing of FG + BG effects (and the
                              children of any `apply` groups).
    add_styleground         — add a parallax, custom Lua effect (e.g.
                              2.5DHelper/VoidBg), or an `apply` group to FG
                              or BG. Auto-creates Style/Foregrounds/
                              Backgrounds if missing.
    remove_styleground      — remove an effect by index.
    update_styleground      — merge property changes into an existing effect
                              (set keys, or pass null to clear them).

No new runtime dependencies.

loenn-mcp v2.0.0
================

Celeste map editor for AI agents — now with procedural generation, seeded
randomness strategies, pattern library extraction, and GameBanana map
ingestion.


WHAT'S NEW IN v2.0
------------------

4 new generation MCP tools (22 tools total):

  build_pattern_library   — scan local .bin maps and extract room patterns
                            into a reusable JSON library (PCG/patterns.json)

  generate_room_from_pattern
                          — generate a new room using patterns from the
                            library with a chosen strategy and seed.
                            Strategies: balanced, exploration, challenge,
                            speedrun.
                            Model profiles: creative, deterministic,
                            architect.

  validate_room           — check a room for playability problems (missing
                            spawn, no floor, entity out-of-bounds, etc.)

  ingest_external_map     — download maps from external URLs (.bin, .zip, or
                            GameBanana mod pages), extract patterns into the
                            library, and store attribution metadata.
                            Requires confirm_download=True as an explicit
                            opt-in; attribution string is recommended.

New module:  loenn_mcp/pcg.py
  - Room pattern extraction (size class, entity density, tile motifs, tags)
  - Pattern library load/save/merge (JSON, deduplicated by SHA-1 hash)
  - Seeded random.Random generation (reproducible with seed + profile)
  - Strategy-based tile grid and entity generation
  - Room validation helper (validate_room_structure)

No new runtime dependencies — stdlib only (urllib, zipfile, hashlib, json).


INSTALL
-------
  pip install loenn-mcp==2.0.0


FULL TOOL LIST (22 tools)
-------------------------
  Map Reading    — list_maps, read_map_overview, read_room, get_room_tiles
  Map Editing    — add_entity, remove_entity, set_room_tiles, add_room,
                   remove_room, create_map
  Entity Catalog — list_entity_definitions, get_entity_definition,
                   list_trigger_definitions, list_effect_definitions
  Analysis       — analyze_map, visualize_map_layout, preview_map_section
  Rendering      — render_map_html
  Generation     — build_pattern_library, generate_room_from_pattern,
                   validate_room, ingest_external_map


CONNECT TO GITHUB COPILOT (VS Code)
-------------------------------------
Add to .vscode/mcp.json:

  {
    "servers": {
      "loenn-mcp": {
        "type": "stdio",
        "command": "python",
        "args": ["-m", "loenn_mcp.server"],
        "env": { "LOENN_MCP_WORKSPACE": "${workspaceFolder}" }
      }
    }
  }

loenn-mcp v1.0.0
================

Celeste map editor for AI agents — read, edit, analyze, and preview Celeste
.bin map files directly from GitHub Copilot, Claude Desktop, or any MCP
client. No Loenn installation required.


INSTALL
-------
  pip install loenn-mcp


WHAT'S INCLUDED
---------------
- 18 MCP tools across 5 categories:
    Map Reading    — list_maps, read_map_overview, read_room, get_room_tiles
    Map Editing    — add_entity, remove_entity, set_room_tiles, add_room,
                     remove_room, create_map
    Entity Catalog — list_entity_definitions, get_entity_definition,
                     list_trigger_definitions, list_effect_definitions
    Analysis       — analyze_map, visualize_map_layout, preview_map_section
    Rendering      — render_map_html (interactive HTML, zoom/pan/search)

- celeste_bin.py — pure-Python .bin parser, full read/write round-trip
                   (no Loenn or Everest required)

- preview_map.py — one-command HTML map preview that opens in the browser

- Works with maps created by Loenn or Ahorn
